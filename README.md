# Vedant Tripathi Portfolio Website

A single-file personal portfolio website for Vedant Tripathi, focused on backend engineering, AI systems, and production-grade infrastructure work.

## Overview

This project is a static portfolio site built in plain HTML, CSS, and JavaScript. It presents Vedant's background, technical stack, work experience, leadership roles, featured projects, education, and contact links in a high-contrast animated layout.

## Features

- Full-screen hero section with animated profile card
- Custom particle background and cursor effects
- GSAP-powered scroll reveals and motion effects
- Dedicated sections for:
  - About
  - Tech stack
  - Work experience
  - Leadership & community
  - Selected projects
  - Education & certifications
  - Contact
- External links for GitHub, LinkedIn, and email

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- [GSAP](https://gsap.com/) + ScrollTrigger
- Google Fonts

## Running Locally

Because this is a static site, no build step is required.

1. Clone the repository
2. Open `vedant-tripathi-portfolio.html` directly in your browser

If you prefer serving it locally, you can use any simple static server.

## Customization

To adapt this portfolio for your own use, update the content inside `vedant-tripathi-portfolio.html`, including:

- Name and headline
- About section
- Skills and tools
- Experience and leadership entries
- Project descriptions and links
- Education details
- Contact information
